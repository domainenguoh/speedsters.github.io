Cool Research Project

How does the complexity of a project's database schema correlate with GitHub activity?

Group Members: Rani Maklada, Neil Hamza, Jeries Shomaly, Bashar Yousef

Project Description

We aim to produce a report outlining the correlation between the complexity of the database schema and the level of activity (commits and issues) in Java and Python projects on GitHub. This report will provide insights into how the complexity of a project's database schema might impact the maintenance and management of the project.
